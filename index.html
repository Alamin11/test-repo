Git basic commands

        Step1: Create a destined folder and open terminal there 
        Step2: Use "git init" command to initialize git 
        Step3: Use "git add ." to stage all the files to be added to git repo
        Step4: Now commit all the files using "git commit -m 'commit message' "
        Step5: So with the above commands all our files are ready to send in a remote repo. As per our convenience we will create a new repository on github
        Step6: Meanwhile we can change our branch master to main using "git branch -M main"
        Step7: Now add the remote repository with our local repo using "git remote add origin remoteRepoURL"
        Step8: Finally its time to send or local repo to remote repo . To do so use "git push -u origin main"
